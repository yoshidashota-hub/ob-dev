# Projects

学んだことを実践するプロジェクトを管理します。

## プロジェクトの条件

- **期限がある**: 1週間〜3ヶ月程度
- **具体的な成果物**: 何かを作る・完成させる
- **学習の実践**: 学んだことを使う

## 例

- Todoアプリを作る
- 技術ブログ記事を書く
- オープンソースにコントリビュート
- ポートフォリオサイト制作

## プロジェクトファイルの構成

各プロジェクトには以下を含める:
- 目標
- タスクリスト
- 進捗メモ
- 学んだこと
- 成果物へのリンク

完了したら振り返りを書いて、学びを `Knowledge` に移動しましょう。
