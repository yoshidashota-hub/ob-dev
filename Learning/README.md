# Learning

学習プロセスを管理するフォルダです。

## 構成

### In-Progress/
今現在学習中のトピック。未完成でもOK。

**例**: React学習、Python基礎、英語文法 など

### Mastered/
十分に理解・習得したトピック。

**移動タイミング**:
- 自分の言葉で説明できる
- 実際に使えるようになった
- 人に教えられるレベル

## ワークフロー

```
Inbox → Learning/In-Progress → Learning/Mastered → Knowledge
```

学習が完了したら、エッセンスを `Knowledge` フォルダに整理すると良い。
