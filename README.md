# 📚 Learning Knowledge Base

学習ノート・知識管理のためのObsidian Vault（ハイブリッド型）

## 📁 ディレクトリ構成

```
/
├── +Inbox/              # 📥 新規メモ・未整理
├── Daily/               # 📅 デイリーノート
├── Learning/            # 🎓 学習プロセス
│   ├── In-Progress/    # 学習中
│   └── Mastered/       # 習得済み
├── Knowledge/           # 💡 知識ベース
│   ├── Concepts/       # 概念・理論
│   ├── HowTo/          # ハウツー・手順
│   └── Examples/       # 実例・サンプル
├── Projects/            # 🎯 実践プロジェクト
├── Index/               # 🗂️ インデックス・MOC
├── Attachments/         # 📎 画像・ファイル
└── Templates/           # 📋 テンプレート
```

## 🚀 はじめ方

1. **[Index/HOME.md](Index/HOME.md)** を開く
2. 新しいメモは `+Inbox/` に作成
3. テンプレートを使ってノートを作成

## 💡 基本ワークフロー

```
1. メモする    → +Inbox/
2. 学習する    → Learning/In-Progress/
3. 習得する    → Learning/Mastered/
4. 整理する    → Knowledge/
5. 実践する    → Projects/
```

## 📝 テンプレート

- **学習ノート.md** - 学習内容をまとめる
- **デイリーノート.md** - 日々の記録
- **プロジェクト.md** - プロジェクト管理
- **概念メモ.md** - 概念・理論の整理
- **HowTo.md** - 手順・やり方

## ⚙️ Obsidian設定のおすすめ

### 1. テンプレートの設定
- 設定 → コアプラグイン → テンプレート
- テンプレートフォルダ: `Templates`

### 2. デイリーノートの設定
- 設定 → コアプラグイン → デイリーノート
- 保存先: `Daily`
- テンプレート: `Templates/デイリーノート.md`

### 3. 新規ノートの保存先
- 設定 → ファイルとリンク
- デフォルトの保存先: `+Inbox`

### 4. 添付ファイルの保存先
- 設定 → ファイルとリンク
- 添付ファイルのフォルダパス: `Attachments`

## 🔗 Git連携

このVaultはGitで管理されています。

**Obsidian Gitプラグイン**を使うと自動バックアップ可能:
- 自動コミット: 5-10分ごと
- 自動プッシュ: 10-15分ごと
