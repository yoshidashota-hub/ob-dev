# AWS Infrastructure Complete Guide

AWS を活用したインフラ構築の実践ガイド。

## 概要

このガイドでは、AWS の主要サービスを使ったインフラ構築から、QuickSight によるデータ可視化、本番運用まで学びます。

## 対象読者

- AWS の基本を学びたい方
- インフラを自分で構築したい開発者
- QuickSight でデータ分析したい方

## 学べること

- AWS の主要サービスと使い分け
- サーバーレスアーキテクチャ
- QuickSight によるBI/分析
- コスト最適化と監視
