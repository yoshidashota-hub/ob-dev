# Performance Guide

Web アプリケーションのパフォーマンス計測と改善の実践ガイド。

## 概要

このガイドでは、フロントエンドからバックエンド、インフラまでの全レイヤーでのパフォーマンス最適化手法を学びます。

## 対象読者

- パフォーマンス改善に取り組む開発者
- Core Web Vitals を改善したいフロントエンド開発者
- レスポンス時間を短縮したいバックエンド開発者

## 学べること

- Core Web Vitals と計測手法
- フロントエンド最適化（React/Next.js）
- バックエンド最適化（API/DB）
- インフラレベルの最適化
- 継続的なパフォーマンスモニタリング
