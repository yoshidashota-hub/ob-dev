## MongoDB の特徴

- **ドキュメント指向**: JSON ライクな柔軟なデータ構造
- **スケーラブル**: シャーディングで水平スケール
- **高速**: インメモリ処理とインデックス
- **柔軟なスキーマ**: スキーマレスで素早い開発

## 対象読者

- NoSQL に興味がある方
- 柔軟なデータモデルを必要とするプロジェクト
- リアルタイムアプリケーション開発者
