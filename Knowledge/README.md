# Knowledge

整理された知識ベースです。すぐに参照できる形でまとめます。

## 構成

### Concepts/
概念・理論・原則など抽象的な知識

**例**:
- オブジェクト指向とは
- RESTful APIの原則
- デザインパターン

### HowTo/
具体的な手順・やり方

**例**:
- Dockerコンテナの起動方法
- Gitでブランチを作成する手順
- React Hooksの使い方

### Examples/
実例・サンプルコード・ケーススタディ

**例**:
- useEffectの実装例
- API通信のサンプルコード
- エラーハンドリングの実例

## ポイント

- **原子的**: 1ファイル = 1つの知識
- **リンク**: 関連する知識を `[[リンク]]` でつなぐ
- **自分の言葉**: コピペではなく理解して書く
